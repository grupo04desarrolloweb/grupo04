Contenido del Curso

¿Qué aprenderás?

Aprenderás a diseñar y construir sitios web modernos usando HTML, CSS, Bootstrap y buenas prácticas de accesibilidad.

¿Qué es?

El frontend se refiere a la parte visible de un sitio web o aplicación con la que los usuarios pueden interactuar directamente. Es la cara del sitio, ubicada en el lado del cliente.

¿Para qué sirve?

Se utiliza para referirse a parte del desarrollo web que involucra todo aquello que el usuario ve, es decir todas las partes del desarrollo que sirven para que las aplicaciones se comuniquen con el usuario.
